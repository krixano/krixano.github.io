Background

I've always wanted to try to make a text adventure. However, I wanted to do something very different from everything else, so I decided to make it an open-world sandbox text adventure, inspired by games like Minecraft, Terraria, Infinimine, etc.

The goals are to make an infinite open-world sandbox that allows for land modification, crafting, multi-player interaction, and building all within a text environment that uses varied language to elevate the writing that describes the world.

The game will also have a natural language parser that will hopefully be as advanced, if not more advanced, as classic text-based games like Zork. My experience with Linguistics and language creation (conlanging) has helped a lot in allowing me to think about how this could be done.
